#安装方法
本课设运用typescript开发，所以要全局安装typescript

npm install typescript -g

还需要angular-cli构建工具

npm install @angular/cli -g

web-front前端为angular-cli构建，启动方式和官网一样
即：
npm install
ng serve

teamflow-end为后端server目录

启动方式：

npm install

tsc -w

进入dist目录
node app.js
